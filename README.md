# ben-pelletier.com
Static single page website leveraging html and css preprocessors.

**Built With**
- Webpack ( module build system )
- Pug ( html preprocessor )
- Stylus ( css preprocessor )
- Skeleton CSS
- jQuery
- ColdFusion


## Build Setup

**Requires Node.js 6+**

``` bash
# install dependencies
npm install

# build for production
npm run build

```
